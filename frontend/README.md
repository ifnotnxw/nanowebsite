# НаноФарм — фронтенд

Статика сайта: раздаётся бэкендом с **http://localhost:3000** при запуске `npm start` из папки `backend/`.

## Запуск без бэкенда

Можно открыть `index.html` через Live Server или другой локальный сервер из папки `frontend/`. Тогда API по умолчанию будет запрашиваться с **http://localhost:3000**; если бэкенд не запущен, используются локальные mock-данные.

Чтобы отключить обращение к API (только mock), в `index.html` перед подключением `app.js` добавьте:
```html
<script>window.__NF_API_BASE = "";</script>
```

## Структура

- `index.html` — главная страница
- `app.js` — логика приложения, роутинг, запросы к API
- `styles.css` — стили
- `pages/` — HTML-фрагменты страниц (каталог, партнёры, новости и т.д.)
- `img/` — логотипы и изображения партнёров
